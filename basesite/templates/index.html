Keyboard Shortcuts

  • SHIFT + / - Show keyboard shortcuts
  • g then n - New task
  • g then t - Task list
  • g then h - Home
  • 1-9 - Select task
  • ↑/↓ or j/k - Navigate tasks
  • Enter - View selected task
  • e - Edit selected task
  • c - Complete selected task
  • d/x - Delete selected task
  • Esc - Deselect / Cancel
